5 Complete Ground Connection

Connect the breadboard ground rail (where all LED cathodes meet) to the GND pin on NodeMCU using a black jumper wire. This creates a common ground for all LEDs.

Plug a Micro USB cable into the NodeMCU USB port. Connect the other end to a 5V power source (phone charger, computer USB port, or power bank). The NodeMCU has a built-in voltage regulator that converts 5V to 3.3V for the ESP8266 chip.

7 Upload Code

Use Arduino IDE to upload your traffic light program to the NodeMCU. Make sure you select NodeMCU 1.0 (ESP-12E Module) as your board and the correct COM port.

‚ö†Ô∏è Critical Safety & Assembly Notes

  • LED Polarity is CRUCIAL: LEDs only work in one direction. Wrong polarity = LED won't light up
  • Always use resistors: Never connect LEDs directly to GPIO pins without a current-limiting resistor
  • Check pin mapping: D1=GPIO5, D2=GPIO4, D5=GPIO14 (not the same as D numbers!)
  • Power before programming: Always connect power before uploading code to prevent errors
  • Double-check connections: One wrong wire can prevent the entire circuit from working
  • Use a breadboard: Makes testing and troubleshooting much easier than soldering

üí° Pro Tips for Success

  • Use colored wires: Match wire colors to LED colors for easier troubleshooting
  • Test one LED at a time: Connect and test each LED individually before connecting all three
  • Breadboard organization: Keep components aligned in rows for a cleaner, easier-to-debug circuit
  • Label your wires: Use tape or labels to mark which wire goes to which pin
  • Check resistor values: Use a multimeter or read color bands carefully (Red-Red-Brown-Gold = 220Œ©)
  • Common ground is key: All LED cathodes must share the same ground connection
